# System-Info-Collector 포트폴리오 개선 PRD

## 프로젝트 개요
System-Info-Collector는 시스템의 CPU, 메모리, 디스크, 네트워크 정보를 수집하여 API 형식으로 제공하는 경량 에이전트입니다. 포트폴리오용으로 시각화, 알림, 데이터 저장, 확장성, 보안을 강화하여 엔터프라이즈급 모니터링 솔루션으로 개선합니다.

## 현재 기능
- 시스템 리소스 모니터링 (CPU, 메모리, 디스크, 네트워크)
- Prometheus 메트릭 제공
- REST API 엔드포인트
- SSH 키 등록 스크립트
- 헬스체크 엔드포인트

## 기술 스택
- Go 1.21+
- Prometheus (메트릭)
- Grafana (시각화)
- InfluxDB/TimescaleDB (시계열 DB)
- Redis (캐싱)
- Docker

## 성능 요구사항
- 초당 1000+ 메트릭 수집
- 응답 시간 < 50ms
- 메모리 사용량 < 100MB
- CPU 사용률 < 10%
